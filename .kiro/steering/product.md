# Music Room - Product Overview

Music Room is a React Native mobile application that enables users to create collaborative music experiences through shared playlists and events. The app integrates with Spotify to provide music streaming capabilities and uses real-time features for collaborative interactions.

## Core Features

- **User Authentication**: Secure sign-up/sign-in with email and password
- **Profile Management**: User profiles with avatars, music tastes, and privacy settings
- **Playlist Management**: Create, edit, and share music playlists
- **Event System**: Create music events with voting capabilities for track selection
- **Social Features**: Follow other users, view their profiles and playlists
- **Music Integration**: Spotify API integration for track search and playback
- **Real-time Collaboration**: Live voting and interaction in events

## Target Platform

- Primary: iOS and Android mobile devices
- Built with React Native and Expo for cross-platform compatibility
- Web support available through Expo web

## Key User Flows

1. **Authentication**: Register → Verify → Login → Access main app
2. **Playlist Creation**: Browse music → Add tracks → Share with others
3. **Event Participation**: Join events → Vote on tracks → Collaborative listening
4. **Social Discovery**: Search users → Follow → View shared content
